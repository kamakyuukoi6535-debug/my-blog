# ブログ記事の投稿手順

## 1. 記事を用意する
Claudeに以下を伝えて記事を作成してもらう：
- テーマ・タイトル
- 文体・トーン
- 文字数の目安

---

## 2. コマンドプロンプトを開く
Windowsキー → 「コマンドプロンプト」で検索 → 開く

---

## 3. my-blogフォルダに移動する
```
cd C:\Users\kamak\OneDrive\Desktop\my-blog
```

---

## 4. 記事ファイルを作成する
```
hugo new content posts/ファイル名.md
```
※ファイル名は半角英数字で（例：running-tips.md、investment-2026.md）

---

## 5. GitHubにアップロードする
```
git add .
git commit -m "記事タイトルを入力"
git push origin main
```

---

## 6. ブログを確認する
1〜2分後にブラウザで確認：
```
my-blog-b4d.pages.dev
```
