## パターン1：Claude Codeに依頼する（最も簡単）

### 依頼の仕方

Claude Codeのチャットに以下をそのまま貼り付けて送る：

```
新規でブログ記事を掲載してほしい。

---
title: "記事タイトル"
date: 2026-XX-XXT09:00:00+09:00
draft: false
tags: ["タグ1", "タグ2"]
description: "記事の説明文"
---

ここに本文を書く
```

### 注意事項

- **日付について** : 深夜0時〜8時台に投稿する場合は、日付を前日にする
  - 例：深夜0時に投稿 → `date: 2026-04-26T09:00:00+09:00`（前日の日付）
- **画像を追加したい場合** : 本文中に `{{IMAGE}}` と書いておくと、その位置に画像を挿入できる
- **記事を取り下げたい場合** : 「〇〇の記事を取り下げてください」と伝えるだけでOK

## パターン2：自分で投稿する

### Step 1: フォルダに移動

コマンドプロンプトを開いて以下を入力：

```
cd C:\Users\kamak\OneDrive\Desktop\my-blog
```

### Step 2: 記事ファイルを作成

```
hugo new content posts/ファイル名.md
```

**ファイル名のルール** : 半角英数字とハイフンのみ（例：`my-new-article.md`）

### Step 3: 記事を編集

`C:\Users\kamak\OneDrive\Desktop\my-blog\content\posts\ファイル名.md` を開いて以下の形式で書く：

```markdown
---
title: "記事タイトル"
date: 2026-XX-XXT09:00:00+09:00
draft: false
tags: ["タグ1", "タグ2"]
description: "記事の説明文（検索結果に表示される）"
---

本文をここに書く
```

**必ず確認** : `draft: false` になっているか（`true` のままだと非公開）

### Step 4: 画像を追加する場合

1. 画像ファイルを以下のフォルダにコピー：
   ```
   C:\Users\kamak\OneDrive\Desktop\my-blog\static\images\
   ```

2. 記事本文に以下を書く：
   ```markdown
   ![画像の説明文](/images/ファイル名.jpg)
   ```

### Step 5: GitHubにプッシュ

```
git add .
git commit -m "記事タイトル"
git push origin main
```

### Step 6: 確認

1〜2分後にブラウザで確認：  
https://run-pull-up-think.com

## トラブルシューティング

### 記事が表示されない（404エラー）

→ **日付が未来になっている可能性**  
深夜0〜8時台に投稿した場合、日付を前日に変更して再プッシュ

```
# hugo.tomlの日付を前日に変更後
git add .
git commit -m "fix: 日付修正"
git push origin main
```

### git pushでエラーが出る

→ インターネット接続を確認してから再実行

### Cloudflare Pagesのビルドが失敗する

→ https://dash.cloudflare.com の「Workers & Pages」→「私のブログ」→「デプロイメント」でエラー内容を確認
